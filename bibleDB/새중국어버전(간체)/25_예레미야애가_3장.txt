3	1	耶和华忿怒的杖 
3	2	他领我，使我行在黑暗中，不行在光明里。 
3	3	他真是终日不停地反手攻击我。 
3	4	他使我的肌肤衰残，折断我的骨头。 
3	5	他筑垒围困我，使毒害和艰难环绕我。 
3	6	他使我住在黑暗之处，好像死了许久的人一样。 
3	7	他筑墙围住我，使我不能逃出去；他又加重我的钢链。 
3	8	甚至我哀求呼救的时候，他也掩耳不听我的祷告。 
3	9	他用砍凿好的石头堵塞我的道路，他使我的路径曲折。 
3	10	他像熊埋伏着，又像狮子在藏匿的地方，等候攻击我。 
3	11	他把我拖离大路，把我撕碎，弃我于荒野。 
3	12	他拉开了他的弓，立我作箭靶子。 
3	13	他把他箭囊中的箭，射进我的肺腑。 
3	14	我成了众民讥笑的对象，他们终日以我为歌嘲讽我。 
3	15	他使我饱尝苦菜，饱享苦堇。 
3	16	他用沙石使我的牙齿破碎，把我践踏在灰尘中。 
3	17	你使我失去了平安，我已忘记了福乐是什么。 
3	18	所以我说：“我的力量已消失了，我从耶和华所得的盼望也没有了。” 
3	19	哀求怜悯回忆起我的困苦飘流，就像是苦堇和毒草。 
3	20	每逢我的心想起往事，我的心就消沉。 
3	21	但我的心一想起下面这件事，我就有指望。 
3	22	耶和华的慈爱永不断绝，他的怜悯永不止息。 
3	23	每天早晨都是新的；你的信实多么广大！ 
3	24	我心里说：“耶和华是我的业分，所以，我必仰望他。” 
3	25	仰望救恩耶和华善待等候他的和心里寻求他的人。 
3	26	安静等候耶和华的救恩，是多么的美好！ 
3	27	人在幼年时就负轭，是多么的美好！ 
3	28	他要无言独坐，因为这是耶和华加在他身上的。 
3	29	他要把自己的口埋于尘土中，或者还有盼望。 
3	30	他要让人打他的脸颊，要饱受凌辱。 
3	31	主必不会永远丢弃人。 
3	32	他虽然使人忧愁，却必照着他丰盛的慈爱施怜悯。 
3	33	因为他心里本不是要人受困苦，或是要世人愁苦。 
3	34	人把地上所有被囚的，都践踏在脚下， 
3	35	或在至高者面前，屈枉正直， 
3	36	或在诉讼的事上颠倒是非，主不都看见吗？ 
3	37	除非主命定，谁能说成，就成了呢？ 
3	38	或祸或福，不都是出于至高者的口吗？ 
3	39	人活在世上，因自己的罪受惩罚，为什么发怨言呢？ 
3	40	悔改归向耶和华我们要检讨和省察自己的行为，然后归向耶和华。 
3	41	我们要向天上的　神，诚心举手祷告： 
3	42	“我们犯罪悖逆，你并不赦免。 
3	43	你被怒气笼罩着，你追赶我们，杀戮我们，毫不顾惜。 
3	44	你用密云把自己遮蔽起来，以致我们的祷告不能达到你那里。 
3	45	你使我们在万族中，成了渣滓和废物。 
3	46	我们所有的仇敌，都张开口攻击我们。 
3	47	我们遭遇的，只是恐惧、陷阱、毁坏和灭亡。” 
3	48	我的眼泪像江河流下，都因我的人民的灭亡。 
3	49	我的眼泪涌流不停，总不止息， 
3	50	直到耶和华垂顾，从天上关注。 
3	51	因眼见我城中众民的遭遇，我的心很痛苦。 
3	52	那些无故与我为敌的人追捕我，像追捕雀鸟一样。 
3	53	他们把我投在坑中要结束我的性命，又把石头拋在我身上； 
3	54	水淹过我的头，我说：“我要死了！” 
3	55	求　神施行拯救耶和华啊，我从坑的最深处呼求你的名， 
3	56	你曾经垂听我的声音，现在求你不要掩耳不听我求救的呼声。 
3	57	我每逢呼求你的时候，你就靠近我，说：“不要惧怕！” 
3	58	主啊！你为我的案件申辩，赎回了我的性命。 
3	59	耶和华啊，你看见了我的冤屈，求你为我主持公道。 
3	60	你已看见了他们种种的仇恨，以及所有害我的阴谋。 
3	61	耶和华啊！你已听见了他们的辱骂，以及所有害我的计谋； 
3	62	你也听见了那些起来攻击我的人嘴里的话，以及他们终日要攻击我的企图。 
3	63	你看，他们或坐下或起来，我都成了他们歌唱嘲笑的对象。 
3	64	耶和华啊！求你按着他们手所作的，报应他们！ 
3	65	求你使他们的心顽梗，愿你的咒诅临到他们。 
3	66	求你在烈怒中追赶他们，从耶和华管治的普天之下除灭他们。
